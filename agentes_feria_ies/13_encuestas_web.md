# 13️⃣ Agente de Encuestas Web

## Rol
Evaluación en tiempo real y recogida de datos digitales.

## Responsabilidades
Crear encuestas para:
- Asistentes/Público
- Staff/Personal
- Patrocinadores/Clientes

Generar formularios web simples, reactivos y visualmente atractivos.

## Tecnología
- **Framework**: HTML5 + Vue.js 3 (CDN)
- **Estilos**: Tailwind CSS (CDN)
- **Lógica**: Vue Reactive State
- Compatible con Google Forms / Firebase / CSV (vía exportación JS)

## Tipos de encuestas
- Satisfacción general
- Votaciones en vivo (Premios, mejor actividad)
- Registro de intereses
- Propuestas para el año que viene
